# Shashank Jagtap - Portfolio Website

## Overview
This is a personal portfolio website for Shashank Jagtap, showcasing professional experience, skills, and projects. The website serves as a comprehensive digital resume and portfolio.

## Features
- **Professional Profile**: Information about my background, skills, and expertise
- **Project Showcase**: Highlights of completed projects with descriptions
- **Experience Timeline**: Chronological display of professional experience
- **Skills Section**: Visual representation of technical and soft skills
- **Contact Information**: Ways to connect with me professionally

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages (for hosting)

## Deployment
The website is automatically deployed using GitHub Pages from the main branch of this repository.
